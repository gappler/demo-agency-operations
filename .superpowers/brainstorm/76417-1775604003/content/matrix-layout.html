Key Elements

Capacity Column
Hours used / billable target, mini bar chart, percentage. Color-coded: red >115%, gray 85–115%, green <70%
Hour Cells
Click any cell to edit hours. Empty cells show "—" with a subtle hover state. Cell background tints match the person's capacity color.
Department Groups
Collapsible rows per department with aggregate capacity. Click to expand/collapse. Shows dept-level utilization.
Over-Allocation Warning
When an edit pushes someone past target, an inline confirmation appears: "Nico will be at 141% — confirm?" with suggested alternatives.
